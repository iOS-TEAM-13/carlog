**앱 구성 : 보라색 하이라이트 → 기능 핵심**
> 
> - 로그인/회원가입 → `Firebase` 사용
> - MyCarPage (MainPage) 내 차 점검 목록
> - HistoryPage (주행기록 저장 / 주유 내역 저장)
> - CommunityPage (기본적인 CRUD)
> - MapPage (주유소 보여주기)
> - MyPage

### ⚙️ Tech Stack

- iOS 15 버전으로 개발

- SnapKit
- Firebase Auth
- Firebase FireStore
- URLsession
- MapKit
- UserDefaults
